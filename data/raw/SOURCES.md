# Data Sources

This project uses public, reproducible data sources. Raw files are stored in this folder so the analysis can run offline after setup.

## Historical World Cup Data

- Fjelstul World Cup Database: https://github.com/jfjelstul/worldcup
- Files used: `matches.csv`, `team_appearances.csv`, `tournaments.csv`, `qualified_teams.csv`, `teams.csv`, `tournament_standings.csv`, `host_countries.csv`
- Purpose: historical men's World Cup matches, team appearances, stages, final standings, hosts, and confederation metadata.

## International Results

- Mart Jurriaans international results dataset: https://github.com/martj42/international_results
- File used: `results.csv`
- Purpose: international match results from 1872 onward. The project uses this only for pre-tournament form before the start date of each World Cup.

## 2026 World Cup Field

- OpenFootball 2026 World Cup groups: https://github.com/openfootball/worldcup/blob/master/2026--usa/cup.txt
- FIFA qualified-team and group pages used for verification:
  - https://www.fifa.com/en/articles/groups-how-teams-qualify-tie-breakers
  - https://www.fifa.com/de/articles/wm-2026-qualifizierte-teams
- Secondary cross-check:
  - https://www.china.org.cn/2026-04/01/content_118413831.shtml

The 2026 table is a preview of the 48-team field and groups. It does not include any 2026 World Cup match results.
